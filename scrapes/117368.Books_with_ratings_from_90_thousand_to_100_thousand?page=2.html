When I started doing more active maintenance work on these lists a year and a half ago, my idea was that I would vote for everything I'd read/wanted to read and use the leftover votes for maintenance. Going beyond that, removing my genuine votes in order to add more newly graduated books, would be fine for a while but not forever. Well - for a while looks like it's turning into forever in the 100k-150k list, so I'm going to take a step back here.
If anyone votes for a book on the next list and just needs help removing it from this one, please leave a comment and I'll be happy to help. If anyone plans to spam the list with that bright yellow book that seems to pop up on every single listopia list (or anything else that clearly doesn't fit), don't even think about it. And if I end up having leftover votes on the next list again, or a book I want to vote for needs graduating, I'll most likely be back every now and then, just not as often and as regularly as I have been.
